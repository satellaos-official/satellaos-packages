# SatellaOS DEB Creator Tool

A simple tool for creating Debian (`.deb`) packages through an easy-to-use graphical interface.

---

## ✨ Features

* Create Debian (`.deb`) packages quickly
* User-friendly PyQt6 interface
* Lightweight and easy to use
* Compatible with Debian-based distributions

---

## 🖥️ Launch from GUI

```bash
python3 deb-creator.py
```

---

## 🚀 Build from CLI

```bash
python3 deb-creator.py --file "$HOME/file.sdcf"
```

---

## 📦 Requirements

### Debian / Ubuntu / SatellaOS

```bash
sudo apt install python3-pyqt6
```

### Arch Linux

```bash
sudo pacman -S python-pyqt6
```

### Fedora / Red Hat

```bash
sudo dnf install python-pyqt6
```
